# Practical 04: Aggregate Functions and Grouping

## Objective
Use aggregate functions along with `GROUP BY` and `HAVING` clauses to retrieve summarized data from the database.

## Steps Performed
1.  **Aggregation**: Used `COUNT()`, `SUM()`, `AVG()`, `MIN()`, and `MAX()` to analyze library inventory.
2.  **Grouping**: Grouped books by `Author_ID` and `Publisher_ID` to see distribution.
3.  **Filtering**: Used `HAVING` to filter groups (e.g., showing only authors with more than 5 books).
