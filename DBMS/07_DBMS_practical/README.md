# Practical 07: Stored Procedures / Functions with Cursors

## Objective
Create and execute stored procedures/functions using cursors to manipulate and retrieve data from the database.

## Steps Performed
1.  **Stored Procedure**: Created a procedure to update book quantities based on issue records.
2.  **Cursors**: Used a cursor within a procedure to iterate through the `Book` table and calculate a discount price or status for a report.
3.  **Functions**: Implemented a function to check if a specific book is in stock.

## Files in this Portfolio
- `Procedures_Cursors.sql`: PL/SQL or T-SQL scripts for procedures and functions.
