# Practical 06: Subqueries

## Objective
Write and execute subqueries to retrieve data from one table based on results from another.

## Steps Performed
1.  **Nested Subquery**: Selected student details who have issued at least one book.
2.  **Correlated Subquery**: Found books that have a quantity higher than the average quantity of all books.
3.  **IN / NOT IN / EXISTS**: Used these operators to filter records based on set membership derived from subqueries.
