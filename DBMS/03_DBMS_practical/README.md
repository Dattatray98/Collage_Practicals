# Practical 03: DML, DCL, and TCL Operations

## Objective
Design and run SQL queries to demonstrate:
- **Data Manipulation (DML)**: `INSERT`, `UPDATE`, `DELETE`.
- **Access Control (DCL)**: `GRANT`, `REVOKE`, `ROLE`.
- **Transaction Control (TCL)**: `COMMIT`, `ROLLBACK`, `SAVEPOINT`.

## Steps Performed
1.  **DML**: Updated student emails and deleted old issue records. Used operators like `BETWEEN` and `LIKE`.
2.  **DCL**: Created a user and granted specific permissions on the library tables.
3.  **TCL**: Performed a multi-step update using `SAVEPOINT` and `ROLLBACK` to ensure atomic transactions.
