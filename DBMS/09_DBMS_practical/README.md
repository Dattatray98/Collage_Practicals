# Practical 09: CRUD Operations using MongoDB

## Objective
Design and implement basic Create, Read, Update, and Delete (CRUD) operations using MongoDB. Use the `save()` method and logical operators where necessary.

## Steps Performed
1.  **Installation & Connection**: Connected to MongoDB shell/compass.
2.  **Create**: Inserted documents into a `students` collection using `insertOne` and `insertMany`.
3.  **Read**: Used `find()` with filters and projection to retrieve specific student data.
4.  **Update**: Used `updateOne` and `updateMany` with `$set` and `$inc` operators.
5.  **Delete**: Used `deleteOne` and `deleteMany` to remove records.

## Files in this Portfolio
- `MongoDB_CRUD.js`: JavaScript commands for MongoDB shell.
